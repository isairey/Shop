## 📌 Descripción

**Botanero Ventas** es un sistema de punto de venta desarrollado para facilitar la gestión de negocios como:

- 🍽 Restaurantes  
- 🍺 Bares / botaneros  
- 🌮 Puestos de comida  
- 🛍 Negocios de alimentos y bebidas  

Permite controlar ventas, pedidos, productos y clientes de forma rápida y eficiente.

---

## 🚀 Características

- 🧾 Registro de ventas en tiempo real  
- 🍔 Gestión de productos y menú  
- 🪑 Control de mesas y pedidos  
- 👥 Administración de clientes  
- 💰 Corte de caja  
- 📊 Reportes de ventas  
- ⚡ Interfaz rápida y fácil de usar

## 💻 Instalación

### 1. Clonar el repositorio
```bash
git clone https://github.com/isairey/botanero-ventas.git
cd botanero-ventas
2. Configurar la base de datos
Crear una base de datos en MySQL
Importar el archivo database.sql
3. Configurar conexión

Editar archivo:

config.php

Colocar tus credenciales:

$host = "localhost";
$user = "root";
$password = "";
$db = "botanero";
4. Ejecutar el proyecto

Usar servidor local:

XAMPP
WAMP
Laragon

Abrir en navegador:

http://localhost/botanero-ventas
