# 🌌 Acerca del proyecto

**Botanero Ventas** es una plataforma POS moderna diseñada para optimizar la administración y operación de negocios gastronómicos.

La plataforma permite gestionar:

- 🍽 Ventas en tiempo real
- 🪑 Control de mesas
- 🍔 Gestión de productos y menú
- 📦 Inventario
- 👥 Clientes y usuarios
- 💰 Corte de caja
- 📊 Reportes administrativos
- 🧾 Generación de tickets

El sistema fue desarrollado pensando en:

- ⚡ Rapidez operativa
- 🧠 Automatización de ventas
- 📱 Interfaz intuitiva
- 🔒 Gestión segura
- 🚀 Escalabilidad empresarial

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/Shop.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd Shop
```

---

## 3️⃣ Configurar base de datos

Crear una base de datos en MySQL e importar:

```bash
database.sql
```

---

## 4️⃣ Configurar conexión

Editar archivo:

```bash
config.php
```

Agregar credenciales:

```php
$host = "localhost";
$user = "root";
$password = "";
$db = "botanero";
```

---

## 5️⃣ Ejecutar servidor local

Usar cualquiera de los siguientes:

- XAMPP
- WAMP
- Laragon

---

## 🌐 Abrir aplicación

```bash
http://localhost/botanero-ventas
```
